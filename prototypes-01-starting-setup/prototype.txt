JavaScript uses "Prototypical Inheritance"

The class Syntax is basically just "syntatic sugar" 

Constructor Functions & Prototypes power JavaScript Objects

구축하는 모든 생성자 함수는 특별한 프로토타입 특성이 있다. 이는 이것에 기초하여 만들어진 객체에 추가되지 않습니다. 함수 본문의 일부가 아니고 함수 객체의 특성이기 때문입니다. 함수 = 객체. 모든 객체는 프로토타입을 가집니다.

프로토타입 객체를 폴백(fallback) 객체로 생각하면 됩니다.
